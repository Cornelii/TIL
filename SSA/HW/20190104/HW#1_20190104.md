## 2. 다음 unix 명령어들에 대하여 간략하게 기술하세요.
- 2-1. ls

디렉토리에 포함된 파일들의 목록을 보여준다.


- 2-2. cd

Change directory의 약자. 디렉토리의 위치를 이동할 때 사용한다.  **Tip:** `.` :현재 디렉토리, `..` : 이전디렉토리


- 2-3. mkdir

Make directory의 약자. 새로운 디렉토리를 생성할 때 사용한다.


- 2-4. touch 

파일을 조작할 때 사용하는 명령어이며, 파일을 생성하거나 파일의 수정시간을 업데이트할 때 사용한다.
