### 1. Linkage between HTML and CSS
* inline   
* internal
* external


```html

<h2 style="color:red;">inline</h2> 

<head>
    <style>
        h1{
            color:blue;
            font-size:50px;
        }

    </style>
</head>
<body>
    <h1>internel</h1>
</body>

<!-- External -->

<head>
    <link href="style.css" rel="stylesheet"/>
</head>


```
#### Remarks! each property should be separated by semicolon ; not comma ,
