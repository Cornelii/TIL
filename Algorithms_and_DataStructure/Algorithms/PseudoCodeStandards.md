# Pseudocode Standard

It helps developer to focus on the logic without destraction by details of language syntax.


**Remarks: Logic must be decomposed to the level of a single loop or decision**

There is no universal "Standard".


## Basic structure
FOR-ENDFOR, IF-ENDIF,WHILE-ENDWHILE, IF_THEN_ELSE_ENDIF
REPEAT_UNTIL

## Common Action Keywords

* Input: READ, OBTAIN, GET
* Output: PRINT, DISPLAY, SHOW
* Compute: COMPUTE, CALCULATE, DETERMINE
* Initialize: SET, INIT
* ADD one: INCREMENT, BUMP
* etc: CALL, BEGIN-EXCEPTION
