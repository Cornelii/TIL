# Array

## I. feature of Constrainted Array?!

When Moving in a 2D array is constrained, the coordinates of points can be decided with other reduced factor.

**example**
From left most at top to right most at bottom. only right an down directions are allowed.
On the condition, The number of Step from (0,0) starting point, and number of column or row can determine some specific point.

This is useful to make DP memoization table with more than two points on 2D Array.
